# ☕ JavaNinja — Batismo de Java (Java 10x)

Este repositório contém meus estudos e práticas realizadas durante o curso **Batismo de Java**, da comunidade Java 10x.

O objetivo deste projeto é consolidar **fundamentos sólidos da linguagem Java**, documentando cada conceito aprendido de forma prática e organizada.

---

## 🎯 Objetivo do Repositório

- Reforçar a base em Java
- Criar hábito de estudo contínuo
- Organizar o aprendizado por níveis
- Manter um histórico real da minha evolução na linguagem

Este projeto faz parte da minha **transição de carreira da área de suporte/infraestrutura para desenvolvimento Java**.

---

## 🧩 Estrutura do Projeto

### 📂 Nível Básico
Conceitos iniciais explicados de forma simples e prática:

- Variáveis e tipagem de dados
- Tipos primitivos e não primitivos
- Entrada de dados com Scanner
- Estruturas condicionais (if/else, switch, ternário)
- Laços de repetição (for, while, do-while)
- Arrays (unidimensionais, multidimensionais)
- Conceitos de memória e Garbage Collector
- Desafios práticos
